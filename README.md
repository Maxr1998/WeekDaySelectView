# WeekDaySelectView
Custom view allowing you to list and select weekdays in a grid.

### Download from Maven
```gradle
repositories {
    maven {
        url  "https://dl.bintray.com/maxr1998/maven"
    }
}

dependencies {
    implementation 'de.Maxr1998.android:weekdayselectview:1.0'
}
```
